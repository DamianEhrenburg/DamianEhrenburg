# Damian Ehrenburg

Junior Python developer focused on Telegram bots, desktop applications and small automation tools.

I build personal projects using Python, PyQt, SQLite, Telegram Bot API, external APIs and Google services. I also experiment with Godot/GDScript and interactive horror prototypes.

## Technical Portfolio

### Librokeeper
Desktop application for managing a home library.

**Stack:** Python, PyQt, SQLite, Google Books API

Features:
- book list with local SQLite storage;
- adding, editing and deleting books;
- search by title/author;
- integration with Google Books API;
- desktop UI built with PyQt.

### Moodcast
Telegram bot for movie and TV recommendations based on the user's mood.

**Stack:** Python, Telegram Bot API, Hugging Face, REST API, JSON

Features:
- mood analysis from text messages;
- movie and TV recommendations;
- integration with an NLP model from Hugging Face;
- external API usage for content data.

### AeroNet
Telegram bot for collecting customer applications.

**Stack:** Python, Telegram Bot API, Google Sheets API

Features:
- step-by-step application form;
- user data confirmation and editing;
- saving applications to Google Sheets.

**Stack:** Godot 4, GDScript, UI, localization
